Design concept

The design concept of this style guide is to use a purple theme website.
Using a clean and spacious approach. Also tried to use rem sizes where applicable
so the design will look better.

To setup a Panel use, a div with class="panel" and if you want specific color scheme
add the following classes: positive, negative, neutral, function, neutraldarker

Same approach with creating a Message, use a div with class="msg" and if you want
specific color scheme add the following classes: error, warning, or from panel color scheme.

Buttons follows same design, choose class="btn" and if you want specific color scheme
add color scheme you want.
